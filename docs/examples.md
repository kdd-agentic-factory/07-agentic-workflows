# Examples

Example files in `examples/` show the input payloads that can be submitted to the orchestrator.

## create-new-feature.input.json
Creates a telemetry anomaly detector feature in 06-kdd-data-pipelines.

## generate-paper-section.input.json
Generates a paper section about the Race Command Center architecture.

## pre-grand-prix.input.json
Prepares the Jerez 2026 race weekend with weather forecast and baseline setup.

## run-setup-what-if.input.json
Simulates a rear rebound and engine map change on FP2 at Jerez.

## validate-autoskill.input.json
Validates a candidate AutoSkill for promotion to generate_session_summary.
